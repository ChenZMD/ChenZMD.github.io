# 全球资金流动可视化 · AI量化交易Agent

基于 D3.js + Topojson 的全球资金流动与经济数据可视化平台。
集成信息分层采集框架（L0/L1/L2/L3），实现从原始数据到交易信号的完整链路。

## 主要功能

### 1. 地图可视化 (chart.html)
- 世界地图国家渲染
- 资金流动粒子动画（60fps）
- 缩放/拖拽交互
- 国家详情面板
- 经济指标展示

### 2. 项目仪表盘 (dashboard.html)
- 项目执行进度追踪
- 实时市场数据展示
- 数据源连通性状态
- 快捷操作入口

### 3. 全流程清单 (checklist.html)
- 7大阶段26步骤进度追踪
- 本地存储自动保存
- 交互式设计

## 五类底层数据系统

```
📊 宏观经济数据 → 海关/航运/PMI/社融/汇率
🏭 产业运行数据 → 电力/新能源/机器人产业链
📢 重要事件数据 → 政策文件/产业公告/技术突破
🌤️ 天气周期数据 → 台风/极端气温/港口停运
📈 二级市场数据 → 板块异动/连板梯队/资金流向
```

## 信息分层采集框架

按"离原始来源的距离"分层采集：
- **L0** 原始层：原文、原始数据、一手材料
- **L1** 专业层：分析框架化后的产物
- **L2** 传播层：被简化/口号化后的版本
- **L3** 反馈层：受众接收后的反应

## 技术栈

- 前端：原生 HTML5 + CSS3 + JavaScript (ES6+)
- 可视化：Canvas 2D + SVG
- 后端：Node.js (数据收集)
- 部署：GitHub Pages

## 快速开始

```bash
# 本地预览
npx serve .

python -m http.server 8080
```

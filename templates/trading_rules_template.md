## 二、周期判断规则

### 2.1 月线级别趋势判断
```
上涨月线定义: 收盘价 > MA5月线 且 MA5 > MA10 > MA20
下跌月线定义: 收盘价 < MA5月线 且 MA5 < MA10 < MA20
震荡月线定义: 均线交织，无明确方向
```

### 2.2 周期位置判定表
| 周期位置 | K线形态 | 成交量特征 | 操作策略 |
|----------|---------|-----------|----------|
| 底部构筑期 | 长期缩量横盘，波动率极低 | 历史地量 | 观察等待，不参与 |
| 突破启动期 | 放量突破关键压力位 | 地量后首次放量 | 试探性建仓(2-3成) |
| 上升趋势期 | 高点、低点依次抬高 | 温和加仓 | 持有加仓(3-5成) |
| 顶部构筑期 | 长上影线，量能异常放大 | 天量天价 | 分批减仓至2成以下 |
| 下降趋势期 | 高点、低点依次降低 | 持续缩量 | 空仓观望 |

### 2.3 时间周期规律
- **月初效应**: 每月前5交易日通常流动性较好
- **月末效应**: 最后3交易日基金调仓，波动加大
- **季度末**: 资金面紧张，避免高杠杆
- **重要会议前后**: 两会/中央经济工作会议/美联储议息会议前降低仓位

## 三、逆向判断规则

### 3.1 高错误率反向指标
| 指标 | 逻辑 | 反向操作 |
|------|------|----------|
| 散户贪婪指数 > 80% | 散户高位接盘 | 卖出 |
| 股神遍地/爆款基金 | 市场过热信号 | 减仓至3成以下 |
| 营业部开户数暴增 | 增量资金见顶 | 暂停加仓 |
| 券商策略一致看多 | 预期过于一致 | 保持警惕 |
| 估值分位数 > 90% | 历史高位区间 | 逐步退出 |

### 3.2 情绪极值反转规则
```
极度恐慌(VIX>30/千股跌停) → 逐步建仓(逆向买入)
极度贪婪(VIX<10/千股涨停) → 逐步减仓(逆向卖出)
```

## 四、板块联动规则

### 4.1 产业链传导链
```
上游(原材料) → 中游(制造) → 下游(消费)
   ↓               ↓            ↓
价格上涨        成本压力      利润压缩
   ↓               ↓            ↓
上游盈利改善    中游分化      下游承压
   ↓               ↓            ↓
资金先炒上游    再炒中游      最后下游
```

### 4.2 关键传导路径
| 传导链 | 信号起点 | 滞后时间 | 标的选择 |
|--------|----------|----------|----------|
| 出口复苏链 | 集装箱运价上涨 | 1-2月 | 制造业/设备 |
| 能源链 | 原油价格上涨 | 2-4周 | 化工/交运 |
| 科技链 | 费城半导体指数 | 同步 | A股半导体 |
| 消费链 | 社零数据回升 | 1-3月 | 白酒/家电 |
| 政策链 | 政策文件发布 | 当周 | 受益板块 |

### 4.3 跨板块共振规则
- 单板块信号: 无效，不操作
- 双板块联动: 弱信号，观察确认
- 三板块以上共振: 高信号，可操作
- 四板块全面共振: 趋势确认，可重仓

## 五、假突破识别规则

### 5.1 放量突破后的假突破特征
- [ ] 突破后3日内回落至压力位下方
- [ ] 突破当日收长上影线(上影>实体2倍)
- [ ] 突破量能不足前高的1.5倍
- [ ] 大盘整体不稳时个股突破

### 5.2 利好出货识别
- [ ] 利好公布前已大涨(预期提前反映)
- [ ] 利好公布后放量滞涨/高开低走
- [ ] 龙虎榜显示机构净卖出
- [ ] 大小非借利好减持

## 六、真趋势启动共振规则

### 6.1 趋势启动四重确认
```
第一重: 技术面 - 突破关键均线系统(5/10/20/60日)多头排列
第二重: 资金面 - 主力资金连续3日净流入>板块总额3%
第三重: 基本面 - 业绩/政策/景气度确实改善
第四重: 共振面 - 三板块以上联动确认
```

### 6.2 共振强度评分标准
| 共振维度 | 因子数量 | 得分 |
|----------|----------|------|
| 技术面 | 3个以上指标确认 | +3 |
| 资金面 | 主力+北向+融资均流入 | +3 |
| 基本面 | 政策+业绩+景气共振 | +2 |
| 情绪面 | 市场情绪配合(非过热) | +1 |
| 跨板块 | 3板块以上联动 | +3 |
| **满分** | | **12** |

**操作阈值:**
- ≥9分: 重仓(6-8成)
- 6-8分: 中等仓位(3-5成)
- 4-5分: 轻仓试探(1-2成)
- <4分: 观望

## 七、止损止盈规则

### 7.1 止损规则
| 买入依据 | 止损条件 | 止损幅度 |
|----------|----------|----------|
| 技术位突破 | 跌破突破位 | -5% ~ -7% |
| 基本面买入 | 基本面逻辑破坏 | -10% |
| 消息驱动 | 消息兑现后回落 | -3% ~ -5% |
| 情绪跟风 | 情绪退潮 | -5% |

### 7.2 止盈规则
| 涨幅阶段 | 操作 | 说明 |
|----------|------|------|
| +10% | 减仓1/4 | 锁定部分利润 |
| +20% | 再减1/4 | 移动成本至安全 |
| +30% | 再减1/4 | 留底仓看趋势 |
| +50%以上 | 用移动止盈 | 跌破MA10清仓 |

### 7.3 时间止损
- 买入后3天无明显方向: 减半
- 买入后7天亏损: 清仓
- 买入后横盘超过14天: 退出(资金效率)

## 八、仓位管理算法

### 8.1 仓位计算公式
```
基础仓位 = 账户总资金 × 风险系数 × 信号强度
风险系数 = 最大可接受亏损 / 止损幅度
信号强度 = 共振评分 / 满分
单票上限 = 账户总资金 × 20%
板块上限 = 账户总资金 × 30%
```

### 8.2 不同市场状态的仓位上限
| 市场状态 | 仓位上限 | 持仓数量 |
|----------|----------|----------|
| 明确上升趋势 | 80-90% | 5-8只 |
| 震荡市 | 40-60% | 3-5只 |
| 明确下降趋势 | 10-20% | 1-2只 |
| 极端行情 | 空仓 | 0 |
